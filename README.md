# Advice Gen 
🔗 **Live Demo:** https://advice-generator-sigma-one.vercel.app/  

A simple React app that generates random advice on button click. Fully responsive and optimized for all screen sizes.  

## 🚀 Features  
✅ Random advice generation  
✅ Clean, minimal UI  
✅ Fully responsive design  
✅ **100/100 Lighthouse performance score**  

## 🛠 Tech Stack  
- **React.js** – Frontend framework  
- **Tailwind CSS** – Styling  
- **JavaScript (ES6+)** – Logic handling  

## 📌 How to Use  
1. **Download ZIP** from the **Code** button and extract it.  
2. Open the folder in **VS Code**.  
3. Open the **VS Code terminal** (`` Ctrl + ` `` or **View > Terminal**).  
4. Move to the `frontend` folder:  
   ```bash
   cd frontend
   ```  
5. Install dependencies:  
   ```bash
   npm install
   ```
6. Start the app:  
   ```bash
   npm run dev
   ```
